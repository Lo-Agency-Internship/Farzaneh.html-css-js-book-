1.What is HTML? HyperText Markup Language (HTML) is the set of markup symbols or codes inserted into a file intended for display on the Internet. The markup tells web browsers how to display a web page's words and images. 2.What is a basic structure of an HTML document? This language consists of a series of elements which tell the browser how to display the content and facilitate website creation. The basic structure of an HTML document consist of header, body, paragraphing and etc. 3.What are the tags/elements that you've learned so far? The html tag represents the root of an HTML document. The html tag is the container for all other HTML elements. These are some examples ehich I have learned so far : html, body, title, head, em and etc.
